Crocodile Kostenkalkulator - Embed Anleitung

Um den Kostenkalkulator auf Ihrer Website einzubetten, fügen Sie bitte den folgenden Code an der gewünschten Stelle ein:

<iframe 
    src="https://savings.crocodile.health"
    style="width: 100%; height: 1200px; border: none; overflow: hidden;"
    title="Crocodile Kostenkalkulator"
></iframe>
Wichtige Hinweise:
  • Die minimale empfohlene Breite beträgt 800px
  • Die minimale empfohlene Höhe beträgt 1200px
  • Stellen Sie sicher, dass Ihre Website HTTPS verwendet
  • Der Kalkulator ist responsiv und passt sich der Breite des iframes an
